# CHANGELOG for tetrarc

Releases (and pre-releases) may be tagged here.
The CHANGELOG acts as a set or ReleaseNotes to describe what has changed with each 'release'

# v0.7.1 - 13 Apr 2026

* Modify tables to define an rclist for each TestBook.
* Modify results tables to have a place for the appropriate RC for this result (default(noRC) is base)
* Add version info in About page
* Fix some problems handling the ReleaseBlocking checkbox
* Modify URL parsing to include RC name (or base) in URL path where needed

# v0.7.0 - 11 Apr 2026

* Make Admin button on navbar a toggle between Admin (edit) and normal views.
* Also can now view detailed Test Results
* Handle getting and displaying total number of 'Admin Pass' results

# v0.6.0 - 08 Apr 2026

A number of changes to reach basic functionality level, including Add/Del/Edit options for
Basic Test definitions
